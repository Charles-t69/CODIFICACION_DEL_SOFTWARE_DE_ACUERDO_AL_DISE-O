## 📦 Instalación General

```bash
# Clonar repositorio
git clone https://github.com/tu-usuario/CODIFICACION_DEL_SOFTWARE_DE_ACUERDO_AL_DISEÑO.git

# Navegar al taller específico
cd Taller_3_Express_MySQL

# Instalar dependencias
npm install

# Configurar variables de entorno
cp .env.example .env

# Ejecutar servidor
npm start
